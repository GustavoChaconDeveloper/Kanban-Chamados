🧩 Sistema de Gestão de Chamados (Kanban)

Sistema de chamados de suporte com visual em Kanban, focado em organização, agilidade e controle do fluxo de atendimento.

A ideia surgiu da necessidade de ter algo mais flexível no dia a dia, sem depender de ferramentas com muitas limitações ou custos elevados.

🚀 Sobre o projeto

O sistema permite que, ao abrir um chamado via formulário, ele já entre automaticamente no fluxo de atendimento como um card.

A partir daí, o time consegue acompanhar tudo de forma visual, movimentando os cards entre as etapas conforme o andamento.

Foi desenvolvido com base em ferramentas do mercado, como o Pipefy, mas com foco em ter mais liberdade de uso e evolução.

⚙️ O que já tem
Abertura de chamados via formulário
Cards organizados em colunas (Kanban)
Movimentação com drag and drop
Visualização das informações do chamado
Controle de tempo de espera
Filtros para facilitar o dia a dia
Painel de BackOffice
🧠 Como funciona na prática

O usuário abre o chamado → vira um card → o suporte assume → movimenta conforme resolve.

Simples, direto e funcional.

🛠️ Tecnologias
Frontend: React, HTML, CSS, JavaScript
Backend: PHP / Node.js
Banco: MySQL / PostgreSQL
